DIV

>> The div tag defines a division or a section in an HTML document.
>> The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
>> The div tag is easily styled by using the class or id attribute.
>> Any sort of content can be put inside the div tag!

H1 - H6

>> The H1 to H6 tags are used to define HTML headings.
>> H1 defines the most important heading.
>> H6 defines the least important heading.

Paragraph

>> The p tag defines a paragraph.
>> Browsers automatically add a single blank line before and after each p tag element.

Image tag

>> The img tag is used to embed an image in an HTML page.
>> Images are not technically inserted into a web page; images are linked to web pages. The img tag creates a holding space for the referenced image.
>> The img tag has two required attributes:

1. src - Specifies the path to the image
2. alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed

a tag (Anchor)

>> The a tag defines a hyperlink, which is used to link from one page to another.
>> The most important attribute of the "a" element is the href attribute, which indicates the link's destination.
>> By default, links will appear as follows in all browsers:

1. An unvisited link is underlined and blue
2. A visited link is underlined and purple
3. An active link is underlined and red

Strong tag (Bold)

Tis tag is used to define text with strong importance. The content inside is typically displayed in bold.

Emphasize tag (Italic)

This tag is used to define emphasized text. The content inside is typically displayed in italic.

Underline tag

For underlining a text or a line, span tag is used and underline style property is applied for it.
The syntax is
< span style="text-decoration: underline;">Underlined text </span >

i-frame

>> The HTML i-frame tag specifies an inline frame.
>> An inline frame is used to embed another document within the current HTML document.

Comment

You can add comments to your HTML source by using the following syntax:
<!-- HTML Comment -->

Horizontal Rule tag

This tag inserts a single line break. This is a self closing tag.
This is used to separate content (or define a change) in an HTML page.

Break tag

This tag jumps to the new line. This is a self closing tag.

Quotation tag (q tag)

The q tag defines a short quotation.

Mail to tag

This tag uses anchor tag to create a link which connects to the mentioned mail id, to compose a mail with the subject "Hi mate".

<a href="mailto:me@ruwix.com?Subject=Hi%20mate" target="_top">Send Mail</a >

Abbreviation

The abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".
The syntax is:
<abbr> title="Hypertext Markup Language">HTML</abbr>

Video tag

>> The video tag is used to embed video content in a document, such as a movie clip or other video streams.
>> The video tag contains one or more source tags with different video sources. The browser will choose the first source it supports.
>> The text between the <video> and </video> tags will only be displayed in browsers that do not support the video element.
>> There are three supported video formats in HTML:
MP4, WebM, and OGG.

Audio tag

>> The audio tag is used to embed sound content in a document, such as music or other audio streams.
>> The audio tag contains one or more source tags with different audio sources. The browser will choose the first source it supports.
>> The text between the <audio> and </audio> tags will only be displayed in browsers that do not support the audio element.
>> There are three supported audio formats in HTML:
MP3, WAV, and OGG.
